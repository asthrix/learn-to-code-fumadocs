# Module 1 · React Fundamentals

This module transforms your scaffold into a living interface. You will learn JSX
syntax, author functional components, pass props, and apply Tailwind utilities
to create polished UI.

### Ship List

-  Global header with project title and quick actions.
-  Sidebar displaying project shortcuts and filters.
-  TaskCard component that renders core task information.

### Lessons (coming next)

1. Introduction to React
2. Why React?
3. JSX Overview
4. JSX Iteration & Conditional Rendering
5. Components
6. Component Props
7. Component Lifecycle (conceptual view in React 18)
8. Introduction to Hooks
9. Styling & Tailwind
10.   Implementation
11.   Validation

Each lesson follows the playbook: quick theory, paired examples, best practices,
common pitfalls, and TaskFlow Pro integration.
