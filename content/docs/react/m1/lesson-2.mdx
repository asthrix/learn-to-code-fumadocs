## What is a Component?

A React component is a **JavaScript function that returns JSX**. That's it!
Every component follows this pattern:

```jsx
function ComponentName() {
   return <div>{/* JSX goes here */}</div>;
}

export default ComponentName;
```

## Step 2: Write the HomePage Component

Add this code to `HomePage.jsx`:

```jsx title="src/pages/HomePage.jsx"
function HomePage() {
   return (
      <div className='home-page'>
         <header className='hero'>
            <h1>Find Your Perfect Stay</h1>
            <p>Discover amazing places to stay around the world</p>
         </header>

         <section className='listings'>
            <h2>Featured Properties</h2>
            <p>Property listings will go here...</p>
         </section>
      </div>
   );
}

export default HomePage;
```

Let's break down what each part does:

### Component Function

```jsx
function HomePage() {
   // ...
}
```

-  Function name uses **PascalCase** (first letter capitalized)
-  Must return JSX
-  Can contain logic before the return statement

### JSX Return

```jsx
return <div className='home-page'>{/* JSX content */}</div>;
```

-  Wrapped in parentheses for multi-line JSX
-  Must have **one root element** (the outer `<div>`)
-  Uses `className` instead of `class`

### Export Statement

```jsx
export default HomePage;
```

-  Makes the component available to other files
-  Use `default` when exporting a single component per file

## Step 3: Import HomePage into App

Now let's use the HomePage component in App.jsx:

```jsx title="src/App.jsx"
import HomePage from "./pages/HomePage"; // ‚Üê NEW: Import the component

function App() {
   return (
      <div className='app'>
         <HomePage /> {/* ‚Üê NEW: Use the component */}
      </div>
   );
}

export default App;
```

**Key points:**

-  Import path is relative (`./pages/HomePage`)
-  No need for `.jsx` extension
-  Component used as JSX tag: `<HomePage />`
-  Self-closing tag since HomePage has no children

## JSX Rules Recap

<Steps>
  <Step>
    ### One Root Element
    JSX must have a single parent element:
    
    ```jsx
    // ‚ùå Wrong: Multiple root elements
    return (
      <h1>Title</h1>
      <p>Text</p>
    );
    
    // ‚úÖ Correct: Wrapped in single parent
    return (
      <div>
        <h1>Title</h1>
        <p>Text</p>
      </div>
    );
    ```
  </Step>
  
  <Step>
    ### Close All Tags
    All tags must be closed, even self-closing ones:
    
    ```jsx
    // ‚ùå Wrong
    <img src="photo.jpg">
    <input type="text">
    
    // ‚úÖ Correct
    <img src="photo.jpg" />
    <input type="text" />
    ```
  </Step>
  
  <Step>
    ### Use className
    HTML's `class` becomes `className` in JSX:
    
    ```jsx
    // ‚ùå Wrong
    <div class="container">
    
    // ‚úÖ Correct
    <div className="container">
    ```
  </Step>
  
  <Step>
    ### JavaScript Expressions in Curly Braces
    Use `{}` to embed JavaScript:
    
    ```jsx
    const name = "StayScape";
    return <h1>{name}</h1>;
    // Renders: <h1>StayScape</h1>
    ```
  </Step>
</Steps>

## Common Mistakes to Avoid

<Accordions type="single">
  <Accordion id="forgot-import" title="Forgetting the Import" value="forgot-import">
    ```jsx
    // ‚ùå This will cause an error:
    function App() {
      return <HomePage />;  // Error: HomePage is not defined
    }
    
    // ‚úÖ Always import first:
    import HomePage from './pages/HomePage';
    ```
  </Accordion>
  
  <Accordion id="wrong-path" title="Wrong Import Path" value="wrong-path">
    ```jsx
    // ‚ùå Wrong path
    import HomePage from './HomePage'; // Not found!
    
    // ‚úÖ Correct relative path
    import HomePage from './pages/HomePage';
    ```
  </Accordion>
  
  <Accordion id="lowercase-component" title="Lowercase Component Name" value="lowercase-component">
    ```jsx
    // ‚ùå React treats lowercase as HTML tags
    function homepage() {
      return <div>...</div>;
    }
    
    // ‚úÖ Always use PascalCase
    function HomePage() {
      return <div>...</div>;
    }
    ```
  </Accordion>
</Accordions>
