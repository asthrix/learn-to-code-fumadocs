## ğŸ” Troubleshooting Failed Checks

### If Development Server Won't Start

```bash
# Clear cache
rm -rf node_modules/.vite

# Reinstall dependencies
rm -rf node_modules package-lock.json
npm install

# Check Node version
node --version  # Must be 18+
```

### If Linting Fails

```bash
# Fix auto-fixable issues
npm run lint:fix

# Check .eslintrc.json is valid JSON
cat .eslintrc.json | python -m json.tool

# Reinstall ESLint
npm install -D eslint
```

### If Build Fails

```bash
# Check for syntax errors
npm run lint

# Clear build cache
rm -rf dist

# Try clean build
npm run build
```

### If HMR Not Working

```bash
# Restart dev server
# Ctrl+C
npm run dev

# Clear browser cache
# Hard refresh: Cmd+Shift+R

# Check vite.config.js exists
cat vite.config.js
```
