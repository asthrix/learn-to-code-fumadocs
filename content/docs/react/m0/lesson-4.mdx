# Lesson 4: Project Overview

Let's explore the complete application you'll build! By the end of this course,
you'll have a fully functional property booking platform.

---

## The Application: "StayScape"

**StayScape** is a property rental marketplace where users can:

-  Browse available properties
-  Search and filter listings
-  View detailed property information
-  Save favorites
-  Book properties
-  Create and manage accounts

Think Airbnb, but you'll build it from scratch! 🏠
