# Module 2 · State & Events

Module 2 breathes life into TaskFlow Pro. You will use `useState`, controlled
inputs, and event handlers to let users create, update, and remove tasks.

### Ship List

-  Task composer form with validation.
-  Inline editing for titles and descriptions.
-  Delete confirmation flow and status toggles.

transitions.
