# Module 4 · Routes & Navigation

Routing makes TaskFlow Pro feel like a complete application. You will configure
React Router, build nested layouts, and handle navigation state such as active
links.

### Ship List

-  Dashboard, Projects, and Settings pages.
-  Nested task detail routes with params.
-  Protected route scaffolding to support authentication later.

1. Introduction to Navigation
2. Client vs Server Routing

4. React Router Part 1 (setup, Routes, Route)
5. React Router Part 2 (Link, NavLink, navigation hooks)
6. React Router Part 3 (nested routes, params, loaders)
7. Routing Best Practices
8. Implementation
9. Validation
