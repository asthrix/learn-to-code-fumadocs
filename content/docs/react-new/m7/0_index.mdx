# Module 7 · Forms & Authentication

Module 7 adds user accounts. You'll collect credentials securely, validate
inputs with React Hook Form, and guard routes with auth-aware layouts.

### Ship List

-  Sign-up, login, and password reset flows.
-  Client-side validation with helpful error messaging.
-  Protected routes and persisted sessions.
