# Module 6 · State Management

Module 6 scales your application architecture. You will leverage the Context API
for lightweight global state and introduce Redux for complex flows.

### Ship List

-  User preference context (theme, density).
-  Redux store for projects/tasks with selectors.
-  Async actions for syncing API changes.
