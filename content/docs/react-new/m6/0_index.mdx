# Module 6 · State Management

Module 6 scales your application architecture. You will leverage the Context API
for lightweight global state, adopt Zustand for UX preferences, and introduce
Redux for complex workflows.

### Ship List

-  Zustand preferences store (theme, density, keyboard shortcuts).
-  Redux store for projects/tasks with selectors and thunks.
-  Async actions and persistence for syncing API changes.

### Lessons

1. Introduction to State Management
2. State Management in React
3. Prop Drilling
4. `useContext`
5. Zustand Introduction
6. Zustand Selectors
7. Zustand Async Flows
8. Zustand Integration
9. Redux Part 1 (store, actions)
10.   Redux Part 2 (reducers, selectors)
11.   Redux Part 3 (async flows)
12.   Redux Custom Components (connectors)
13.   Redux Best Practices
14.   Implementation
15.   Validation
