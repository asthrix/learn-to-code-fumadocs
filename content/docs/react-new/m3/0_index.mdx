# Module 3 · Effects & Data

You will move from local-only state to synced data. Effects keep your UI in sync
with remote sources, while React Query (optional) streamlines caching and
retries.

### Ship List

-  Task list wired to mock API.
-  Loading, error, and skeleton states for lists.
-  Persistence for CRUD actions with optimistic updates.

### Lessons

1. Introduction to Effects
2. Effects in React
3. `useEffect` Deep Dive
4. `useRef` for DOM and mutable values
5. Introduction to Data Fetching
6. Data Fetching Part 1 (fetch)
7. Data Fetching Part 2 (async/await)
8. Data Fetching Part 3 (caching & retries)
9. Data Fetching Best Practices
10.   Implementation
11.   Validation
