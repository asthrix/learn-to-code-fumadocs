# Module 1: React Setup & First Component

Welcome to your React journey! In this module, you'll learn how to set up a
React development environment and create your first React component.

## Prerequisites

Before we start, make sure you have:

-  **Node.js** (version 14 or higher) installed
-  **npm** or **yarn** package manager
-  A code editor (VS Code recommended)
-  Basic knowledge of JavaScript, HTML, and CSS

## Setting Up Your Development Environment

### Option 1: Create React App (Recommended for Beginners)

The easiest way to get started with React is using Create React App:

```bash
npx create-react-app my-first-react-app
cd my-first-react-app
npm start
```

### Option 2: Vite (Modern Alternative)

For a faster development experience, you can use Vite:

```bash
npm create vite@latest my-react-app -- --template react
cd my-react-app
npm install
npm run dev
```

## Understanding JSX

JSX is a syntax extension for JavaScript that looks similar to HTML. Here are
the key differences:

```jsx
// JSX Example
function Welcome() {
   const name = "React Developer";

   return (
      <div className='welcome-container'>
         <h1>Hello, {name}!</h1>
         <img src='/logo.png' alt='React Logo' />
         <button onClick={() => alert("Hello!")}>Click me!</button>
      </div>
   );
}
```

### JSX Rules to Remember

1. **className instead of class**: Use `className` for CSS classes
2. **camelCase for attributes**: `onClick` instead of `onclick`
3. **Self-closing tags**: `<img />`, `<br />`, `<input />`
4. **JavaScript expressions**: Use `{}` to embed JavaScript

## Creating Your First Custom Component

Let's create a simple greeting component:

```jsx
// src/components/Greeting.js
import React from "react";

function Greeting({ name, age }) {
   return (
      <div
         style={{
            padding: "20px",
            border: "2px solid #4CAF50",
            borderRadius: "8px",
            margin: "10px 0",
         }}
      >
         <h2>Hello, {name}!</h2>
         <p>You are {age} years old.</p>
         <p>Welcome to React!</p>
      </div>
   );
}

export default Greeting;
```

Now use it in your `App.js`:

```jsx
// src/App.js
import React from "react";
import Greeting from "./components/Greeting";
import "./App.css";

function App() {
   return (
      <div className='App'>
         <header className='App-header'>
            <h1>My React Application</h1>
            <Greeting name='Alice' age={25} />
            <Greeting name='Bob' age={30} />
            <Greeting name='Charlie' age={22} />
         </header>
      </div>
   );
}

export default App;
```

## Hands-On Exercise: Personal Card Component

Create a `PersonalCard` component that displays personal information:

```jsx
// src/components/PersonalCard.js
import React from "react";

function PersonalCard({ name, occupation, location, hobbies, avatar }) {
```

## Key Concepts Learned

1. **React Components**: Functions that return JSX
2. **JSX Syntax**: HTML-like syntax in JavaScript
3. **Props**: Data passed to components
4. **Component Composition**: Using components inside other components
5. **Styling**: Inline styles and CSS classes
