## 🎯 What You'll Learn

By the end of this course, you'll be able to:

-  ✅ **Understand HTML document structure** and semantic elements
-  ✅ **Create accessible web pages** following WCAG guidelines
-  ✅ **Build responsive layouts** with modern HTML5 elements
-  ✅ **Implement forms, media, and interactive elements**
-  ✅ **Optimize for SEO** with proper markup structure
-  ✅ **Use modern HTML5 APIs** for enhanced functionality

## 🏗️ Project: Personal Portfolio Website

Throughout this course, you'll build a complete **personal portfolio website**
featuring:

-  **Responsive layout** that works on all devices
-  **Semantic HTML structure** for accessibility and SEO
-  **Contact forms** with proper validation
-  **Media galleries** showcasing your work
-  **Modern HTML5 features** like semantic elements and microdata

## 📚 Course Modules

### Module 1: HTML Foundations

-  Introduction to HTML and the web
-  Document structure and basic elements
-  Text formatting and semantic meaning
-  **Project:** Create your first HTML page

### Module 2: Semantic HTML

-  Understanding semantic vs. presentational markup
-  Header, nav, main, article, section, aside, footer
-  Proper heading hierarchy
-  **Project:** Build the portfolio structure

### Module 3: Forms and Input Elements

-  Form elements and input types
-  Form validation and accessibility
-  Modern input types (email, tel, date, etc.)
-  **Project:** Add a contact form

### Module 4: Media and Embedded Content

-  Images, audio, and video elements
-  Responsive images with srcset
-  Embedding content (maps, social media)
-  **Project:** Create a media gallery

### Module 5: Tables and Data

-  When and how to use tables
-  Table accessibility features
-  Styling table data
-  **Project:** Add a skills/experience table

### Module 6: Advanced HTML5

-  Custom data attributes
-  Microdata and structured data
-  HTML5 APIs overview
-  **Project:** Enhance portfolio with advanced features

## 🔧 Prerequisites

-  **Basic computer skills** - File management and web browsing
-  **Text editor** - VS Code (recommended) or any code editor
-  **Web browser** - Chrome, Firefox, or Safari for testing
-  **Curiosity to learn** - No prior programming experience needed!
