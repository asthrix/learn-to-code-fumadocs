# Props & Composition

StaySense now needs real data on the page. You'll craft `ListingCard`, pass data
with props, and render a small collection in `ListingSection`.

---

```tsx
export type Listing = {
```

## 2. Create the ListingCard Component

Add `src/components/ListingCard.tsx`:

```tsx
import type { Listing } from "../data/listings";

export function ListingCard({ listing }: { listing: Listing }) {
   return (
      <article className='overflow-hidden rounded-2xl border border-white/10 bg-white/5 shadow-lg shadow-black/20 backdrop-blur'>
         <img
            src={listing.imageUrl}
            alt={listing.name}
            className='h-48 w-full object-cover'
            loading='lazy'
         />
         <div className='space-y-3 p-6'>
            <header className='flex items-center justify-between gap-3'>
               <h3 className='text-lg font-semibold text-white'>
                  {listing.name}
               </h3>
               <span className='rounded-full bg-emerald-400/20 px-3 py-1 text-xs font-medium text-emerald-300'>
                  ${listing.pricePerNight}/night
               </span>
            </header>
            <p className='text-sm text-slate-300'>{listing.location}</p>
            <button className='w-full rounded-lg border border-white/20 px-4 py-2 text-sm text-white transition hover:border-emerald-400 hover:text-emerald-200'>
               View details
            </button>
         </div>
      </article>
   );
}
```

---

## 3. Render the Cards in ListingSection

Update `ListingSection`:

```tsx
import { ListingCard } from "./ListingCard";
import { featuredListings } from "../data/listings";

export function ListingSection() {
   return (
      <section id='listings' className='bg-slate-900/60 py-16'>
         <div className='mx-auto flex w-full max-w-5xl flex-col gap-8 px-6'>
            <div>
               <p className='text-xs uppercase tracking-[0.3em] text-emerald-300'>
                  Curated picks
               </p>
               <h2 className='text-2xl font-semibold text-white'>
                  Featured getaway homes
               </h2>
            </div>
            <div className='grid gap-6 md:grid-cols-2 lg:grid-cols-3'>
               {featuredListings.map((listing) => (
                  <ListingCard key={listing.id} listing={listing} />
               ))}
            </div>
         </div>
      </section>
   );
}
```

Props let `ListingCard` stay flexible—swap the `featuredListings` array with API
data later without altering the component itself.
