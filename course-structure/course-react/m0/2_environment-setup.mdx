## Installing Node.js

Node.js includes npm (Node Package Manager), which we'll use to install React
and other dependencies.

### Download & Install

<Tabs items={["macOS", "Windows", "Linux"]}>
   <Tab value='macOS'>
      **Option 1: Official Installer** 1. Visit
      [nodejs.org](https://nodejs.org/) 2. Download the LTS version
      (recommended) 3. Run the installer 4. Follow the installation wizard
      **Option 2: Using Homebrew** ```bash brew install node ```
   </Tab>
   <Tab value='Windows'>
      1. Visit [nodejs.org](https://nodejs.org/) 2. Download the LTS version
      (recommended) 3. Run the `.msi` installer 4. Follow the installation wizard
      5. Restart your computer **Note**: Make sure to check "Add to PATH" during
      installation
   </Tab>
   <Tab value='Linux'>
      **Ubuntu/Debian:** ```bash curl -fsSL
      https://deb.nodesource.com/setup_lts.x | sudo -E bash - sudo apt-get
      install -y nodejs ``` **Fedora:** ```bash sudo dnf install nodejs ```
      **Using nvm (recommended):** ```bash curl -o-
      https://raw.githubusercontent.com/nvm-sh/nvm/v0.39.0/install.sh | bash nvm
      install --lts ```
   </Tab>
</Tabs>

### Verify Installation

Open your terminal and run:

```bash title="Terminal"
node --version
npm --version
```

You should see version numbers like:

```
v20.10.0
10.2.3
```

If you see version numbers, you're all set! If not, restart your terminal or
computer.

## Setting Up VS Code

Visual Studio Code is a free, powerful code editor with excellent React support.

### Install VS Code

1. Visit [code.visualstudio.com](https://code.visualstudio.com/)
2. Download for your operating system
3. Install and launch VS Code

### Essential Extensions

Install these extensions for a better React development experience:

<Accordions>
   <Accordion title='ES7+ React/Redux/React-Native snippets'>
      **What it does**: Provides code snippets for React components **Install**:
      Search "ES7+ React" in VS Code extensions **Benefits**: - Type `rafce` to
      create a functional component - Type `useState` for state snippets - Saves
      time with common patterns
   </Accordion>


<Accordion title='ESLint'>
   **What it does**: Identifies and fixes code problems **Install**: Search
   "ESLint" in VS Code extensions **Benefits**: - Catches bugs before runtime -
   Enforces code style - Suggests best practices
</Accordion>


<Accordion title='Prettier - Code formatter'>
   **What it does**: Automatically formats your code **Install**: Search
   "Prettier" in VS Code extensions **Benefits**: - Consistent code formatting -
   Saves time on styling - Works with team standards
</Accordion>

   <Accordion title='Auto Rename Tag'>
      **What it does**: Automatically renames paired HTML/JSX tags **Install**:
      Search "Auto Rename Tag" in VS Code extensions **Benefits**: - Rename
      opening tag, closing tag updates automatically - Prevents mismatched tags
   </Accordion>
</Accordions>

### VS Code Settings

Configure VS Code for optimal React development:

1. Open Settings (Cmd/Ctrl + ,)
2. Search for "format on save"
3. Enable "Format On Save"
4. Search for "default formatter"
5. Select "Prettier"

Or add to your `settings.json`:

```json title="settings.json"
{
  "editor.formatOnSave": true,
  "editor.defaultFormatter": "esbenp.prettier-vscode",
  "editor.codeActionsOnSave": {
    "source.fixAll.eslint": true
  },
  "javascript.updateImportsOnFileMove.enabled": "always",
  "editor.tabSize": 2
}
```

## Browser Setup

### Install a Modern Browser

Use Chrome or Firefox for the best development experience.

-  **Chrome**: [google.com/chrome](https://www.google.com/chrome/)
-  **Firefox**: [mozilla.org/firefox](https://www.mozilla.org/firefox/)

### Install React Developer Tools

React Developer Tools is a browser extension for debugging React applications.

<Tabs items={["Chrome", "Firefox"]}>
   <Tab value='Chrome'>
      1. Visit the [Chrome Web Store](https://chrome.google.com/webstore) 2.
      Search for "React Developer Tools" 3. Click "Add to Chrome" 4. Confirm the
      installation
   </Tab>

   <Tab value='Firefox'>
      1. Visit [Firefox Add-ons](https://addons.mozilla.org/firefox/) 2. Search
      for "React Developer Tools" 3. Click "Add to Firefox" 4. Confirm the
      installation
   </Tab>
</Tabs>

After installation, you'll see two new tabs in your browser's developer tools:

-  **⚛️ Components** - Inspect React component tree
-  **⚛️ Profiler** - Measure component performance

## Terminal Basics

You'll use the terminal (command line) throughout this course. Here are
essential commands:

### Navigation

```bash title="Terminal"
# Show current directory
pwd

# List files and folders

ls

# Change directory

cd folder-name

# Go back one directory

cd ..

# Go to home directory

cd ~
```

### File Operations

```bash title="Terminal"
# Create a new directory
mkdir my-folder

# Create a new file
touch file.txt

# Remove a file
rm file.txt

# Remove a directory
rm -rf folder-name
```

### Opening Projects

```bash title="Terminal"
# Open current directory in VS Code
code .

# Open specific folder

code my-project
```

## Troubleshooting

<Accordions>
  <Accordion title="Node.js command not found">
    **Solution:**
    - Restart your terminal/computer
    - Check if Node.js is in your PATH
    - Reinstall Node.js
    - On Windows, make sure "Add to PATH" was checked during installation
  </Accordion>

  <Accordion title="Permission errors on macOS/Linux">
    **Solution:**
    ```bash
    # Fix npm permissions
    mkdir ~/.npm-global
    npm config set prefix '~/.npm-global'
    echo 'export PATH=~/.npm-global/bin:$PATH' >> ~/.profile
    source ~/.profile
    ```
  </Accordion>

  <Accordion title="VS Code extensions not working">
    **Solution:**
    - Restart VS Code
    - Check extension is enabled
    - Update VS Code to latest version
    - Reload VS Code (Cmd/Ctrl + R)
  </Accordion>
</Accordions>
